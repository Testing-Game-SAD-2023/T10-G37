installare junit5
apt install junit5
compilare i file .java
javac -d ./target/classes src/main/java/ClientProject/*.java
javac -cp ../usr//share/java/junit-jupiter-api-5.3.2.jar -d ./target/classes src/test/java/ClientProject/*.java

Eseguire i la misurazione


java -jar evosuite-1.0.6.jar -measureCoverage -class ClientProject.Calcolatrice -Djunit=ClientProject.NomeClasseTest -projectCP $PWD/target/classes/ClientProject -Dcriterion=LINE
java -jar evosuite-1.0.6.jar -measureCoverage -class Calcolatrice -Djunit=NomeClasseTest -projectCP /target/classes -Dcriterion=LINE
java -jar evosuite-1.0.6.jar -measureCoverage -class  ClientProject.Calcolatrice -Djunit=ClientProject.NomeClasseTest -projectCP /ClientProject/target/classes -Dcriterion=LINE
